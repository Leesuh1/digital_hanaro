# Clarity Invest Landing Page

AI powered investment education landing page built with only HTML, CSS, and vanilla JavaScript.

## Local preview

Open `index.html` directly in your browser.

## Project structure

- `index.html`
- `style.css`
- `script.js`
- `vercel.json`

## GitHub deployment

1. Create a new GitHub repository.
2. Add the remote:

```bash
git remote add origin <YOUR_GITHUB_REPO_URL>
```

3. Commit and push:

```bash
git add .
git commit -m "Initial landing page"
git branch -M main
git push -u origin main
```

## Vercel deployment

1. Log in to Vercel.
2. Import the GitHub repository.
3. Keep the project as a static site with the root directory set to this folder.
4. Deploy.

After that, every push to `main` will automatically trigger a new Vercel deployment.
